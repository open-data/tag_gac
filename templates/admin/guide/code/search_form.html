Instructions

Each trade agreement has either of a list of covered commodity codes or excluded codes.

There are a few different trade agreement coding systems.

Federal Supply Classification System (FSC)

Common Classification System (CCS)

Central Product Classification (CPC)

The GC developed the Goods and Services Identification System (GSIN) for its commodity code system

The table of commodity codes below was created by creating a crosswalk between the trade codes

and the GSIN codes. We then shortened the list of codes to only those with will be in EPS.


There is a row for each commodity code. You can edit the models here or by clicking the row ID.

Use the search field to search by commodity code. Or you can filter entities by trade agreement coverage.

There are checkboxes on the far left. If you want to delete a row, click the checkbox then under Action select the dropdown option and click Go.


ID: Click on the row ID to edit the information in a row.

CODE [EN]: The commodity code text in english.

CODE [FR]: The commodity code text in french.

COMMODITY TYPE: Goods, Services, or Construction

The rest of the columns are for Canada's trade agreements.

A checkmark indicates that the entity is covered by the trade agreement. No checkmark means the entity is not covered by the trade agreement.
