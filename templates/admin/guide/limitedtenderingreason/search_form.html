Instructions

Each trade agreement has a list of reasons a contracting officer may use to limit tendering.

There is a lot of similarity between each agreement, but there are subtle differences.

This tool is setup so that the limited tendering page is dynamic. If no trade agreements

apply then the page will not appear to the user. When a trade agreement does apply the

user will only be shown only the limited tendering reasons that are applicable for those

trade agreements.


There is a row for each limited tendering reason. You can edit the models here or by clicking the row ID.

Use the search field to search for keywords. Or you can filter limited tendering reasons by trade agreement coverage.

There are checkboxes on the far left. If you want to delete a row, click the checkbox then under Action select the dropdown option and click Go.


ID: Click on the row ID to edit the information in a row.

DESCRIPTION [EN]: The text of the limited tendering reason in english.

DESCRIPTION [FR]: The text of the limited tendering reason in french.

The rest of the columns are for Canada's trade agreements.


A checkmark indicates that that trade agreement and limited tendering reason is valid for that trade agreement.

No checkmark means the entity is not covered by the trade agreement.
